# AI 駐守助手 - 功能與業務說明書

本文件說明 Wafer BI 平台中 AI 駐守助手的業務價值、功能範圍與使用場景。

## 1. 業務價值
AI 駐守助手旨在降低工程師與分析師在海量晶圓數據中檢索資訊的成本。透過自然語言對話，使用者無需具備複雜的 SQL 或報表查詢知識，即可快速獲取關鍵決策資訊。

## 2. 主要功能
- **即時狀態查詢**: 使用者可以直接詢問特定晶圓 (Wafer ID) 或批次 (Lot ID) 的當前測量數值與統計結果。
- **異常分析引導**: AI 能主動識別數據中的變異性 (Std Dev)，幫助使用者定位潛在的製程問題。
- **自然語言搜尋**: 支援如「幫我找出厚度 (Thickness) 異常的晶圓」等模糊查詢。
- **全天候駐守**: 前端界面右下角設有常駐入口，使用者在任何分析頁面皆可隨時呼叫 AI 協助。

## 3. 使用場景範例
- **早會報告**: 快速確認昨日產出的批次是否有數據偏離。
- **異常調查**: 當某片晶圓失效時，詢問 AI 該晶圓在其他參數下的表現。
- **數據探索**: 詢問系統中目前有哪些晶圓的特定參數超出了預設範圍。

## 4. 注意事項
- AI 助手的回答基於資料庫中已向量化的數據，建議定期執行數據同步以確保資訊最新。
- 當系統配置出現問題（如 API 授權失效）時，界面會主動彈出告警，請及時聯繫管理員。
