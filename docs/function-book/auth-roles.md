# 👥 用戶角色與權限說明 (Authentication & Roles)

## 1. 登入體驗
Wafer BI 提供直覺且現代化的登入介面，支援：
*   **快速演示帳號**：提供一鍵填入「演示帳號 (demo01)」的功能，方便快速進入系統展示功能。
*   **個人化註冊**：一般用戶可自行註冊帳號，系統會自動分配為標準用戶群組。

## 2. 角色定義 (RBAC)
系統根據用戶群組（Group）決定可存取的選單與數據範圍：

| 角色名稱 | 權限等級 | 核心功能 | 說明 |
| :--- | :--- | :--- | :--- |
| **管理員 (admin)** | 最高 | 全功能存取、用戶管理 | 負責維護系統、審核用戶與管理機密數據。 |
| **演示帳號 (demo01)** | 高 (Sudo) | 完整業務功能存取 | 具備與管理員相同的業務權限，專為產品展示設計。 |
| **一般用戶 (user)** | 一般 | 報表檢視、數據查詢 | 僅能存取基礎的晶圓分析與看板功能。 |

## 3. 功能過濾與動態選單 (Dynamic Menu System)
系統採用動態選單機制，根據用戶所屬群組 (Group) 與選單 (Menu) 的關聯決定顯示內容：
*   **群組關聯 (Group-Menu Mapping)**：管理員可在後端資料庫中設定特定群組所能看到的 Menu 列表。
*   **前端動態渲染**：登入成功後，API 會回傳該用戶被授權的選單 (如 `lot-overview`, `system-status` 等)，前端會根據此列表動態生成導覽列。
*   **權限層級**：
    *   **管理員 (admin)**：預設擁有所有選單存取權，包含「用戶管理」與「系統狀態」。
    *   **一般用戶**：僅顯示被分配到的基礎分析頁面。

## 4. 管理員控制台 (Admin Console)
管理員具備以下高級操作權限：
1.  **重設用戶密碼**：可直接幫遺忘密碼的用戶重新設定新密碼。
2.  **變更用戶權限**：隨時調整用戶所屬的 Group，即時生效其選單權限。
3.  **用戶生命週期管理**：包含新增帳號與刪除冗餘帳號。
