# 👥 用戶角色與權限說明 (Authentication & Roles)

## 1. 登入體驗
Wafer BI 提供直覺且現代化的登入介面，支援：
*   **快速演示帳號**：提供一鍵填入「演示帳號 (demo01)」的功能，方便快速進入系統展示功能。
*   **個人化註冊**：一般用戶可自行註冊帳號，系統會自動分配為標準用戶群組。

## 2. 角色定義 (RBAC)
系統根據用戶群組（Group）決定可存取的選單與數據範圍：

| 角色名稱 | 權限等級 | 核心功能 | 說明 |
| :--- | :--- | :--- | :--- |
| **管理員 (admin)** | 最高 | 全功能存取、用戶管理 | 負責維護系統、審核用戶與管理機密數據。 |
| **演示帳號 (demo01)** | 高 (Sudo) | 完整業務功能存取 | 具備與管理員相同的業務權限，專為產品展示設計。 |
| **一般用戶 (user)** | 一般 | 報表檢視、數據查詢 | 僅能存取基礎的晶圓分析與看板功能。 |

## 3. 功能過濾機制
系統會根據您的登入身份，自動隱藏或顯示左側導覽選單：
*   **非管理員**：將無法看到「用戶管理 (User Management)」選單。
*   **未登入**：僅能存取登入與註冊頁面，所有 API 請求將受保護。
