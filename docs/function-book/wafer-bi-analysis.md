# 🔬 Wafer BI 分析功能手冊 (Business Logic)

## 1. 核心看板 (Dashboard)
系統提供工業級的晶圓數據分析介面，主要包含以下三個層級：

### 1.1 批次總覽 (Lot Overview)
*   **功能**：檢視特定生產批次 (Lot ID) 的所有晶圓列表。
*   **關鍵指標**：顯示每片晶圓的平均厚度 (Thickness)、應力 (Stress) 等參數的基礎統計。

### 1.2 晶圓詳情 (Wafer Detail)
*   **功能**：點擊特定晶圓可查看詳細的數據分佈。
*   **視覺化**：透過熱力圖 (Heatmap) 或 3D 視圖展示晶圓表面的參數變化。

### 1.3 統計分析與數據報告
*   **功能**：針對特定批次進行趨勢分析。
*   **圖表類型**：包含直方圖 (Histogram)、累計分佈函數 (CDF) 以及控制圖 (Control Chart)。

## 2. 數據更新邏輯
*   **動態生成**：為確保演示時隨時有數據，系統在找不到數據時會自動觸發「數據生成器」，模擬真實生產環境的晶圓分佈。
*   **一致性**：同一批次的數據在重新抓取時保持一致，確保分析結果的可追蹤性。
